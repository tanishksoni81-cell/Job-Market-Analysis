📊 Job Market Analysis Dashboard (Python + SQL + AI)

🚀 Overview

This project analyzes job market trends using a combination of **Python, SQL, and AI-powered dashboard generation**.

It focuses on extracting meaningful insights such as:

* Job demand trends
* Salary distribution
* Skills in demand
* Location-based hiring patterns

The workflow integrates **data cleaning, analytical querying, and automated visualization** to simulate a real-world data analytics pipeline.

🛠 Tools & Technologies

* **Python** – Data cleaning, preprocessing, and EDA
* **SQL** – Data querying and insight extraction
* **ChartGen.ai** – Automated dashboard generation
* **Pandas, Matplotlib, Scikit-learn** (optional extensions)

🔄 Project Workflow

1. **Data Collection**

   * Job dataset with fields like job title, salary, skills, location, and experience

2. **Data Cleaning (Python)**

   * Handled missing values
   * Standardized salary formats
   * Cleaned and normalized skills column
   * Removed duplicates

3. **Exploratory Data Analysis (Python)**

   * Skill frequency analysis
   * Salary trends
   * Role-based demand insights

4. **SQL Analysis**

   * Aggregated job demand by role
   * Salary comparisons
   * Location-based job distribution
   * Experience vs salary trends

5. **AI Dashboard Generation**

   * Uploaded cleaned dataset into ChartGen
   * Generated automated dashboard
   * Refined insights and validated outputs

---

📊 Key Insights

* Python and SQL are among the most in-demand skills
* Software Engineering and Data Analyst roles dominate hiring
* Salary increases with experience but shows non-linear trends
* Remote and metro cities offer competitive compensation
* Hiring trends show time-based fluctuations
